# Pre-Game — video submission (Remotion)

85-second hackathon video for **Pre-Game**, built with [Remotion](https://remotion.dev).

## Storyboard

| Scene | Time | What |
|---|---|---|
| 1. Title | 0:00–0:04 | "Pre-Game" stamp, LIVE HYPE LINE marquee |
| 2. Hook | 0:04–0:10 | "Job interview in 5 minutes. Your hands are shaking. What do you need?" |
| 3. Demo — Interview | 0:10–0:27 | Type prompt → reveal THE INTERVIEW, mantra, energy meters. **Real generated music plays.** |
| 4. Demo — Quitting | 0:27–0:42 | LAST SHIFT, real quitting-track music |
| 5. Demo — First Date | 0:42–0:55 | FIRST DATE, first-date track |
| 6. turbopuffer | 0:55–1:15 | Animated cluster diagram showing live semantic map. Counters tick. Captions explain vector embedding → time-filtered queries → ANN solo-twin search. |
| 7. Close | 1:15–1:25 | "Walk in ready." URL. Credits. |

Total: **1:25** (under 90 s).

## Commands

```bash
npm install           # install Remotion + deps
npm run preview       # live Remotion Studio at localhost:3000
npm run render        # full render → out/pregame.mp4
```

## Audio

`public/audio/*.mp3` are the actual tracks generated by the live ElevenLabs Music API
via the Pre-Game `/api/hype` endpoint. They were pulled from `../public/samples/`
(see `scripts/gen-samples.ts` at the repo root).
