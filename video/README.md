# Pre-Game — TikTok video submission (Remotion)

**58-second vertical TikTok** for **Pre-Game** — built with [Remotion](https://remotion.dev).

- 📱 **1080×1920** (9:16)
- 🎞️ **60 fps** (3480 frames)
- 🔊 Real generated music from the live `/api/hype` endpoint under every demo
- 💬 Baked-in captions (big, bottom-third — readable sound-off)

## Storyboard (viral TikTok pacing)

| Scene | Time | What |
|---|---|---|
| 1. HOOK | 0:00–0:02.5 | "**47 people** are walking into **job interviews** right this second." — number ticks up; stop-the-scroll moment. |
| 2. SETUP | 0:02.5–0:05 | "Introducing **Pre-Game**. The moment right before deserves a soundtrack. **▶ watch this**." |
| 3. DEMO — Interview | 0:05–0:15 | Prompt types in → HYPE button pulses → **THE INTERVIEW** stamps. Real ElevenLabs music plays. Mantra drops. Equalizer reacts. Caption: "your hands are shaking. your brain won't stop." |
| 4. DEMO — Quitting | 0:15–0:24 | Hard cut. Pink palette. **LAST SHIFT**. Caption: "rehearsed for weeks. never done this before." |
| 5. DEMO — First Date | 0:24–0:32 | Cyan/violet palette. **FIRST DATE**. Caption: "sitting in my car. can't get out." |
| 6. turbopuffer | 0:32–0:46 | Cluster viz: 5 moment-type clusters appear across the screen with live counters ticking. Captions: "not just you." → "turbopuffer is mapping them all." → "every moment = a vector. live-clustered." |
| 7. CTA | 0:46–0:58 | "**walk in ready.**" → URL (chunky yellow button) → "which moment are **you** walking into?" → #ElevenHacks credits. |

Total: **0:58** — under 60s for the TikTok retention boost.

## Viral design choices

- **0–2s hook**: number counter + "right this second" hits before the thumb-scroll finishes
- **Pattern interrupt every ~2s**: hard cuts between demos, new color palette each cut
- **Social proof**: the turbopuffer scene literally shows "you're not alone" with real numbers
- **Captions baked in**: readable without sound (most TikTok watchtime is muted)
- **Cliff-hanger close**: "which moment are **you** walking into?" → comments bait
- **Lowercase text** feels native to TikTok, not corporate

## Commands

```bash
npm install           # install Remotion + deps
npm run preview       # Remotion Studio at localhost:3000 — iterate live
npm run render        # renders out/pregame.mp4 — full 60fps 1080x1920 h264
```

Render takes ~15 min on a 16-core machine at concurrency=6.

## Audio

`public/audio/*.mp3` are actual tracks generated by the live Pre-Game `/api/hype` endpoint
